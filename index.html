About This Project

Research Purpose

This project analyzes the relationship between smartphone usage patterns and addiction levels among teenagers. Using machine learning techniques, we can predict addiction levels based on behavioral and demographic factors.

Methodology

The analysis uses a Random Forest regression model trained on 3000 teen records with 25 features including phone usage patterns, sleep habits, academic performance, and mental health indicators.
